# 30-Days-SQL-Micro-Course
SQL query solutions for the 30 Days SQL Micro Course, covering patient records, hospital analytics, expenses tracking, and departmental insights. Ideal for learners and aspiring data analysts to practice real-world healthcare and finance scenarios.

## Contents
The repository includes solutions for:

1. **Total Number of Patients**  
   Query to count all patients in the dataset.

2. **Average Number of Doctors per Hospital**  
   Grouped analysis of doctors across hospitals.

3. **Top 3 Departments with Highest Patients**  
   Ranking departments by patient count.

4. **Hospital with Maximum Medical Expenses**  
   Identifying the hospital with the highest expenses.

5. **Daily Average Medical Expenses**  
   Expense trends by admission date.

6. **Longest Hospital Stay**  
   Calculating patient stay duration using `DATEDIFF`.

7. **Total Patients Treated Per City**  
   Aggregating patient counts by location.

8. **Average Length of Stay Per Department**  
   Department-wise average stay duration.

9. **Department with Lowest Number of Patients**  
   Finding the least busy department.

10. **Monthly Medical Expenses Report**  
    Summarizing expenses month by month.
